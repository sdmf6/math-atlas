## 答案
$\sqrt{10}$ 或 $3\sqrt{10}$
## 解析
分两种情况讨论：①当点$E$在线段$CD$上时，$B, D', E$三点共线，根据 $S_{\triangle ABE}=\dfrac{1}{2}AB\cdot AD=\dfrac{1}{2}BE\cdot AD'$ 可求得$BE=5$，再由勾股定理可得 $BD'=\sqrt{AB^{2}-AD^{2}}=4$ ，进而可计算 $DE=D'E=1$ ，在$Rt\triangle ADE$中，由勾股定理计算$AE$的值；②当点$E$在射线$CD$上时，设$CE=x$，则 $D'E=DE=x+5$ ，$BE=x+1$，由勾股定理可解得$x=4$，进而可计算$DE=9$，在$Rt\triangle ADE$中，由勾股定理计算$AE$的值即可.

$\because \angle E D^{\prime}A = \angle D = \angle A D^{\prime}B = 90^{\circ},$ $\therefore B,D^{\prime},E$ 三点共线， $\because S_{\triangle ABE} = \dfrac{1}{2} AB\cdot AD = \dfrac{1}{2} BE\cdot AD^{\prime},$ $\therefore BE = AB = 5,$ $\because BD' = \sqrt{AB^2 - AD'^2} = \sqrt{5^2 - 3^2} = 4,$ $\therefore DE = D'E = BE - BD' = 5 - 4 = 1;$ $\therefore$ 在 $Rt\triangle ADE$ 中， $AE = \sqrt{AD^2 + DE^2} = \sqrt{3^2 + 1^2} = \sqrt{10}$ ；

$\because \angle AD'B = \angle BCE = 90^{\circ}, AD = BC = AD' = 3, AB = CD = 5,$ $\therefore BD' = \sqrt{AB^2 - AD'^2} = 4,$ 设 $CE = x$ ，则 $DE = DE = x + 5,$ $\therefore BE = D'E - BD' = x + 1,$ $\because CE^2 +BC^2 = BE^2$ ，即 $x^{2} + 3^{2} = (x + 1)^{2},$ 解得 $x = 4,$ $\therefore DE = CD + CE = 5 + 4 = 9,$ $\therefore$ 在 $Rt\triangle ADE$ 中， $AE = \sqrt{AD^2 + DE^2} = \sqrt{3^2 + 9^2} = 3\sqrt{10}$ 综上所述， $AE$ 的值为 $\sqrt{10}$ 或 $3\sqrt{10}$ 故答案为： $\sqrt{10}$ 或 $3\sqrt{10}$
